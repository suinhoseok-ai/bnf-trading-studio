## 1. Netlify 환경변수 `BROKER_ENC_KEY` 추가

이 키는 여러분이 입력한 증권사 API 키를 서버에서 **암호화해서 저장**하는 데 쓰는 "자물쇠 열쇠"입니다. 아무나 못 여는 긴 문자열이면 됩니다.

### 1단계. 암호화 키(긴 문자열) 만들기
아래 셋 중 편한 방법 하나로 **40자 이상**의 무작위 문자열을 만드세요.

- **가장 쉬움**: 키보드를 아무렇게나 길게 눌러 만들기 (영문 대소문자+숫자 섞어 40자 이상)
  예: `k7Qp2mZx9Lw3Rn8Vt5Bc1Dy6Hf4Jg0Ps7Ue2Ma9Kd` (← 예시를 그대로 쓰지 말고 직접 만드세요)
- **Windows PowerShell로 생성**:
  ```powershell
  -join ((48..57)+(65..90)+(97..122) | Get-Random -Count 48 | ForEach-Object {[char]$_})
  ```
  실행 후 나온 48자를 복사하세요.
- **온라인**: "random password generator 64" 검색 → 64자 생성

> ⚠️ 이 문자열은 **한 번 정하면 바꾸지 마세요.** 나중에 바꾸면 이미 저장해둔 증권사 키를 복호화(해독)하지 못합니다. 안전한 곳(메모장·비밀번호 관리자)에 백업해 두세요.

### 2단계. Netlify에 등록
1. [app.netlify.com](https://app.netlify.com) 로그인
2. 배포된 **사이트(bnf-trading-studio 등)** 클릭
3. 상단 메뉴 **Site configuration**(또는 Site settings) 클릭
4. 왼쪽 메뉴에서 **Environment variables** 클릭
5. **Add a variable** 버튼 → **Add a single variable** 선택
6. 입력:
   - **Key**: `BROKER_ENC_KEY` (오타 주의, 대문자 그대로)
   - **Value**: 1단계에서 만든 긴 문자열 붙여넣기
   - Scopes: 기본값(All) 그대로
7. **Create variable** 클릭

### 3단계. 재배포 (중요!)
환경변수는 **다시 배포해야** 반영됩니다.
1. 상단 메뉴 **Deploys** 클릭
2. 오른쪽 **Trigger deploy** ▾ 버튼 클릭
3. **Deploy site** 클릭 (또는 "Clear cache and deploy site")
4. 초록색 **Published** 표시가 뜰 때까지 1~2분 대기

### 확인 방법
배포 후 앱의 **자동매매(⚡)** 페이지에 들어갔을 때, 예전에 보이던
*"⚠️ 서버에 BROKER_ENC_KEY 환경변수가 없어 API 키가 암호화되지 않습니다"* 경고가 **사라지면** 성공입니다.

## 2. 한국투자증권 API 앱 등록 → App Key/Secret 발급 (모의투자 먼저)

자동매매가 실제 증권사와 통신하려면 **App Key**와 **App Secret**이라는 두 개의 열쇠가 필요합니다. 처음에는 **가짜 돈으로 연습하는 "모의투자"** 키부터 만듭니다.

### 준비물
- 한국투자증권 **계좌** (없으면 먼저 개설 — 한국투자증권 앱/홈페이지에서 비대면 개설 가능)
- 한국투자증권 **홈페이지(HTS/MTS) 로그인 아이디**

### 1단계. 모의투자 계좌 신청 (모의투자 키를 만들려면 필수)
1. 한국투자증권 홈페이지 [securities.koreainvestment.com](https://securities.koreainvestment.com) 로그인
2. 검색창에 **"모의투자"** 검색 → **국내주식 모의투자 참가신청**
3. 참가 신청하면 **모의투자 전용 계좌번호**가 생깁니다 (실계좌와 별개, 앞 8자리 + 뒤 2자리 형태)
4. 이 계좌번호를 메모해 두세요 (나중에 앱에 입력).

### 2단계. KIS Developers 가입 & 로그인
1. [apiportal.koreainvestment.com](https://apiportal.koreainvestment.com) 접속
2. 우측 상단 **로그인** → 한국투자증권 홈페이지 아이디로 로그인 (별도 가입 절차가 뜨면 약관 동의)

### 3단계. 앱(App) 신청 → 키 발급
1. 상단 메뉴 **KIS Developers** → **API 신청/관리**(또는 "My App", "앱 관리") 메뉴로 이동
2. **앱 등록**(App 등록/신청) 버튼 클릭
3. 입력 화면에서:
   - **앱 이름**: 아무거나 (예: `BNF자동매매`)
   - **계좌 선택 / 유형**: **모의투자(모의계좌)** 를 선택 ← ★ 처음엔 반드시 모의!
   - 1단계에서 만든 **모의투자 계좌번호**를 연결
   - **서비스(권한)**: 국내주식 주문/시세(잔고조회 등) 체크
4. 신청 완료하면 **App Key**와 **App Secret** 두 값이 발급됩니다.
   - App Key: 보통 짧은 편 (예: `PSxxxxxxxx...`)
   - App Secret: 아주 긴 문자열
5. **두 값을 즉시 복사해서 안전한 곳에 저장**하세요. (특히 Secret은 다시 안 보여줄 수 있습니다)

> 💡 **실전 키는 나중에**: 모의투자로 며칠 정상 동작을 확인한 뒤, 같은 화면에서 이번엔 **실전투자(실계좌)** 로 앱을 하나 더 등록하면 실전용 App Key/Secret이 따로 나옵니다.

### 4단계. 앱(자동매매 페이지)에 입력
1. 배포된 사이트 → **자동매매(⚡)** 페이지
2. **증권사 연결 설정**에서:
   - 증권사: **한국투자증권 (KIS Open API)**
   - 투자 모드: **모의투자 (KIS 모의계좌)** ← ★
   - **App Key**: 3단계에서 복사한 값
   - **App Secret**: 3단계에서 복사한 값
   - **계좌번호(앞 8자리)** / **계좌상품코드(뒤 2자리, 보통 01)**: 모의투자 계좌번호를 나눠서 입력
3. **연결 테스트** 클릭 → *"연결 성공! 예수금 ○○원"* 이 뜨면 정상 (모의투자는 보통 초기 가상자금이 들어있음)
4. **저장** 클릭
5. 아래에서 전략·매수비중 등을 설정하고 **▶ 자동매매 시작**

## 3. 한국투자증권 → 토스증권으로 바꾸는 방법

### ⚠️ 먼저 알아둘 것 (매우 중요)
**토스증권은 현재 개인 투자자용 "공개 매매 API"를 제공하지 않습니다.**
즉, 지금은 **토스증권으로는 실제 자동매매를 할 수 없습니다.** (2026년 7월 기준)

이 앱은 미래에 토스증권이 API를 공개할 것에 대비해, 프로그램 구조(**Broker Adapter 패턴**)만 미리 만들어 두었습니다.
그래서 화면에는 토스증권 선택지가 보이지만, 실제로 연결하면 아래 안내 문구가 뜹니다:

> "토스증권은 아직 개인용 공개 매매 API를 제공하지 않습니다. 한국투자증권(KIS)을 선택하세요.
> (토스증권이 Open API를 공개하면 어댑터 교체만으로 지원 예정)"

### 3-A. 앱에서 증권사를 "바꾸는" 방법 (화면 조작 자체)
증권사 전환은 화면에서 이렇게 합니다 (토스 API가 나오면 이 방법 그대로 사용):
1. **자동매매(⚡)** 페이지로 이동
2. 안전을 위해 실행 중이면 먼저 **⏸ 자동매매 중지** 를 누르세요
3. **증권사 연결 설정**에서:
   - **증권사** 드롭다운을 **`한국투자증권 (KIS Open API)`** → **`토스증권 (공개 API 미제공 — 준비됨)`** 으로 변경
   - 그 증권사에서 발급받은 **App Key / App Secret / 계좌번호**를 입력
4. **연결 테스트** 클릭
   - **지금은** 토스가 API를 제공하지 않으므로 위의 "제공하지 않습니다" 오류가 뜹니다 → 다시 **한국투자증권**으로 되돌리세요.
   - **토스 API가 나온 뒤**라면 여기서 "연결 성공"이 떠야 정상입니다.
5. **저장** → **▶ 자동매매 시작**

> 참고: 증권사나 투자 모드(모의↔실전)를 바꾸면, 이전 증권사의 로그인 토큰 캐시는 자동으로 초기화됩니다. 그래서 바꾼 뒤에는 **연결 테스트**를 다시 한 번 해주는 것이 좋습니다.

### 3-B. 토스증권 API가 실제로 공개되면 해야 할 일 (개발자용)
토스증권이 개인용 Open API를 공개하면, **화면·자동매매 엔진은 손대지 않고** 어댑터 파일 하나만 채우면 됩니다.
1. 토스증권 개발자 포털에서 App Key/Secret(또는 그에 준하는 인증정보) 발급
2. `src/lib/broker/toss.ts` 파일의 각 메서드를 실제 토스 API 호출로 구현
   (현재는 전부 "미지원" 오류를 던지는 껍데기 상태 — `connect`, `getAccount`, `getPositions`, `getOrders`, `getMarketPrice`, `placeBuyOrder`, `placeSellOrder`, `cancelOrder`)
   - 참고 모델: 같은 폴더의 `kis.ts`(한국투자증권 완전 구현)를 그대로 본떠 작성
3. `npm run build`로 확인 후 배포
4. 이후 사용자는 3-A 방법대로 화면에서 증권사만 토스로 바꾸면 됩니다.
   → **엔진·화면 수정 불필요.** 이것이 Broker Adapter 패턴의 목적입니다.

### 3-C. 지금 당장 자동매매를 하고 싶다면
**한국투자증권(KIS)** 을 사용하세요. 토스증권 계좌를 쓰시더라도, 자동매매를 위해서는
별도로 한국투자증권 계좌를 개설하고 KIS Open API로 운용하는 것이 현재로선 유일한 방법입니다.

## 4. 전체 순서 요약

1. **`BROKER_ENC_KEY`** Netlify 등록 → 재배포 (자동매매 페이지 경고 사라짐 확인)
2. **KIS**: 모의투자 신청 → KIS Developers 로그인 → **모의** 앱 등록 → App Key/Secret 발급
3. **자동매매 페이지**: 한국투자증권 + 모의투자 모드로 입력 → 연결 테스트 → 저장 → ▶ 시작
4. **며칠 모의투자로 검증** 후 실전 앱 등록 → 실전 모드 전환 (소액부터)
5. **토스증권**: 현재는 자동매매 불가(공개 API 없음). API 공개 시 `src/lib/broker/toss.ts`만 구현하면 화면에서 전환 가능.

> ⚠️ **주의**: 실전 모드는 실제 돈으로 주문이 나갑니다. 반드시 모의투자로 충분히 확인한 뒤 소액으로 시작하세요.
> 증권사 홈페이지·개발자 포털의 메뉴 이름은 개편에 따라 조금씩 다를 수 있습니다. 막히면 그 화면을 캡처해서 물어보세요.
>
> 본 시스템은 학습·연구용이며, 투자 판단과 그 결과에 대한 책임은 전적으로 사용자 본인에게 있습니다.
